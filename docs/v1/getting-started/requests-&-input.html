Basic Input

You may access all user input with a few simple methods. You do not need to worry about the HTTP verb used for the request, as input is accessed in the same way for all verbs.

Retrieving An Input Value

var name = req.input.get('name');

Retrieving A Default Value If The Input Value Is Absent

var name = req.input.get('name', 'Sally');

Determining If An Input Value Is Present

if (req.input.has('name'))
{
    //
}

Getting All Input For The Request

var input = req.input.all();

Getting Only Some Of The Request Input

var input = req.input.only('username', 'password');

var input = req.input.except('credit_card');

Cookies

All cookies created by the Quorra framework are encrypted and signed with an authentication code, meaning they will be considered invalid if they have been changed by the client. See Cookie Parser middleware for more.

Retrieving A Cookie Value

var value = req.cookies.name;

Attaching A New Cookie To A Response

res.cookie('name', 'tobi', { domain: '.example.com', path: '/admin', secure: true });
res.cookie('rememberme', '1', { expires: new Date(Date.now() + 900000), httpOnly: true });

Old Input

You may need to keep input from one request until the next request. For example, you may need to re-populate a form after checking it for validation errors.

Flashing Input To The Session

req.input.flash();

Flashing Only Some Input To The Session

req.input.flashOnly('username', 'email');

req.input.flashExcept('password');

Retrieving Old Data

req.input.old('username');

Files

Note: The bodyParser middleware included with Quorra only handles JSON and urlencoded form submissions, not multipart. For file methods to work you may need to add additional file parsing middleware to your application.

Retrieving An Uploaded File

var file = req.input.file('photo');

{
    //

Request Information

The Request object provides many methods for examining the HTTP request for your application and extends the Node http request object. Here are some of the highlights.

Retrieving A Request Header

var header = req.header('referer');

Check If The Given type(s) Is Acceptable

 // Accept: text/html
 req.accepts('html');
 // => "html"

Determining If The Request Is Over HTTPS

if(req.secure) {
    //
}

Determine If The Request Is Using AJAX

if(req.xhr) {
    //
}
